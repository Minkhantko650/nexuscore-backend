## Stack

- chat.py - /chat endpoint (RAG with OpenAI)
- database.py - SQLAlchemy engine, SessionLocal, Base, get_db()
- models.py - User table model
- auth.py - (IN PROGRESS) register + login endpoints
- docker-compose.yml - ChromaDB (port 8001) + PostgreSQL (port 5432)
- .env - OPENAI_API_KEY, DATABASE_URL
